#### JSTL
+ Java Standard Tag Library (JSTL)
	+ https://tomcat.apache.org/taglibs.html 에서 라이브러리 다운로드
	+ impl,spec,jstlel 3개의 jar파일을 lib폴더에 복사
+ jsp 소스 안에서 java코드가 많이 작성이 됩니다.
	+ 제어문 : if, for
	+ format 적용할때
	+ String 메소드 사용할 때
	+ <% %> 스크립트릿 대신에 태그를 사용
	
#### EL
+ Expression Language 표현식
+ <%= %> 대신에 사용 ${ }
+ 변수 수식 결과값 출력이 목적
+ Attribute 가져올때 사용
